The data used to generate these plots are simulated outputs from an LSTM model at a daily time step, covering the period from January 1980 to yesterday (assuming the data has been updated). Below is an explanation of the different plot types:

  1. "Daily": Plots the daily streamflow (in cubic meters per second, cms) for the selected station you click on in the Leaflet map.
  2. Plots the daily streamflow (cms) for the current year. Additionally, the daily range of streamflows from all previous years is shown using the following color scheme (adapted from USGS WaterWatch):
    • Above 90th percentile: Much above normal
    • 75th - 90th percentile: Above normal
    • 25th - 75th percentiles: Normal
    • 10th - 25th percentile: Below normal
    • Below 10th percentile: Much below normal
  3. "Daily cumsum (by year)": Plots the cumulative daily streamflow (cms) from the beginning of the calendar year for the current year. Additionally, the cumulative daily ranges of streamflow from all previous years are displayed using the same color scheme as in "Daily (by year).""
